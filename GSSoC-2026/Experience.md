# 🌸 My GSSoC 2026 Journey

## Getting Started

I joined **GirlScript Summer of Code (GSSoC) 2026** to gain practical experience in open-source software development, collaborate with developers, and understand real-world GitHub workflows.

As this was my first significant open-source program, my primary goal was to learn rather than focus only on rankings or the number of contributions.

---

## My Journey

During GSSoC 2026, I explored multiple projects, understood contribution guidelines, and learned how open-source communities collaborate.

I selected issues based on my current skill level, forked repositories, created feature branches, implemented changes, and submitted pull requests for review.

Through this process, I became more confident using Git, GitHub, and contributing to existing codebases.

---

## Skills I Developed

* Git & GitHub Workflow
* Forking and Cloning Repositories
* Branch Management
* Pull Requests
* Issue Tracking
* Open Source Collaboration
* Code Review Process
* HTML, CSS & JavaScript
* Problem Solving

## What I Learned

GSSoC taught me that open source is not only about writing code but also about communication, collaboration, understanding existing projects, and continuously improving through feedback.

This experience strengthened my confidence in contributing to real-world software projects and encouraged me to continue contributing beyond GSSoC.

---

## Looking Ahead

I plan to continue contributing to open-source projects while applying the skills I gained during GSSoC to future internships, cybersecurity projects, and community-driven initiatives.
